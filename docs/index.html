Creation

Navit(oo)n

〜街中を自動で走るぶつからないロボット〜

自律移動ロボットNavit(oo)n (ナビトン)は，人や自転車が行き交う市街地をぶつかること無く自動で走行する． 2025年の大阪万博にて自動でゴミ拾いを行うことを目標として研究中である． 中之島ロボットチャレンジに参加し大阪中之島公会堂の周辺を走行し実証実験を行っている．

VICT(OO)N

ペットボトルを投げるロボット

高専ロボコン2018で近畿地区大会優勝&全国大会特別賞を受賞したロボット.

黒酢ドリンクのペットボトルに消臭ビーズを入れてバネの力で投げている．

さるかにんぽう

風船を割るロボット

高専ロボコン2017で近畿地区大会技術賞&全国大会特別賞を受賞したロボット.

ヤスリがついた矢を投げたり押し付けて風船を速攻で割る．エキシビションで全国最速タイムを出した．
